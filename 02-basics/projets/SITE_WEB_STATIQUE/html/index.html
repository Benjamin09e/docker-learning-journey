Ce site web est hébergé dans un conteneur Docker avec Nginx comme serveur web. L'utilisation de Docker permet une déploiement rapide et une isolation des dépendances.

Le design est entièrement responsive et s'adapte aux mobiles, tablettes et ordinateurs de bureau.
